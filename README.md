# 每日瑜伽打卡工具

一个简单易用的每日瑜伽打卡工具，帮助你养成每日练习的好习惯。

## 功能特点

- ✅ **一键打卡** - 点击按钮即可完成每日打卡
- 📊 **数据统计** - 显示连续打卡天数、累计打卡次数、本月完成率
- 📅 **历史记录** - 查看最近的打卡历史
- 💾 **本地存储** - 数据保存在浏览器本地，无需登录
- 📱 **响应式设计** - 适配手机和电脑浏览器
- 🎨 **美观界面** - 渐变背景和现代化设计

## 使用方法

1. 打开 `yoga-tracker.html` 文件
2. 点击"点击打卡"按钮完成今日瑜伽打卡
3. 查看统计数据，了解自己的坚持情况

## 技术实现

- 纯HTML/CSS/JavaScript实现
- 使用localStorage进行数据存储
- 无需后端服务，完全离线可用
- 自动计算连续打卡天数

## 如何部署

### GitHub Pages部署
1. Fork此仓库
2. 在仓库设置中启用GitHub Pages
3. 选择main分支作为源
4. 访问 `https://[你的用户名].github.io/[仓库名]` 即可使用

### 本地使用
1. 下载 `yoga-tracker.html` 文件
2. 用浏览器打开即可开始使用

## 数据说明
所有打卡数据都保存在浏览器的localStorage中，不会上传到任何服务器。如果想重置数据，可以在浏览器开发者工具中清除localStorage。
